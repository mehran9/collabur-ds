Collabur Design System

The Collabur Design System build is made up of three parts:

1. Granular Storybook Documentation

To view details on design tokens, elements, components, and prototyped views, use your local instance of Storybook by concurrently running 'yarn && yarn storybook', which will load here: http://localhost:9006/

3. Stencil.js built elements and components

To view the local build for Stencil.js, run 'yarn', which will load here: http://localhost:3333/
